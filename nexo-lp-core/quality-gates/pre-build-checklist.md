# Pre-Build Checklist

Run through this checklist before initiating the build process.

## Content Completeness
- [ ] All {{PLACEHOLDER}} values replaced with actual content
- [ ] No placeholder brackets remaining in output
- [ ] Brand name consistent throughout
- [ ] All navigation links point to real sections/pages
- [ ] Contact information is accurate and current

## HTML Structure
- [ ] Valid HTML5 doctype declared
- [ ] Single `<h1>` element per page
- [ ] Heading hierarchy is logical (no skips: h1 > h2 > h3)
- [ ] Semantic HTML tags used (header, nav, main, section, footer)
- [ ] All images have descriptive `alt` text
- [ ] All images have `width` and `height` attributes
- [ ] Below-fold images use `loading="lazy"`

## Meta / SEO
- [ ] `<title>` tag present and 50-60 characters
- [ ] Meta description present and 150-160 characters
- [ ] Open Graph tags (og:title, og:description, og:image, og:url)
- [ ] Twitter Card tags present
- [ ] Canonical URL set correctly
- [ ] Robots meta tag configured

## Assets
- [ ] Images optimized and compressed
- [ ] Favicon provided
- [ ] Google Fonts using `display=swap`
- [ ] Font preconnect hints added

## Accessibility
- [ ] Color contrast meets WCAG 2.1 AA
- [ ] Focus indicators visible
- [ ] Form inputs have associated labels
- [ ] Language attribute set on `<html>`
- [ ] Skip-to-content link present (optional)
